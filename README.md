# Astryvra Engine

## About the Project

**Astryvra Engine** is a straightforward, cross-platform **C++ game engine** focused on **3D rendering** and **game development**.

This project aims to provide a **lightweight and easily extensible foundation** for aspiring and hobbyist developers who seek a deep understanding of 3D graphics principles, engine architecture, and game logic. It leverages modern C++ standards and rendering APIs to ensure efficient and visually appealing results.
